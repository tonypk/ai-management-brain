# Culture Packs Reference

9 culture packs control communication style per-employee.

| Culture | Directness | Hierarchy | Key Rule |

## Override Rule

Culture overrides mentor when they conflict. Examples:

- **Dalio + Filipino employee** -> private feedback (not public radical transparency)
- **Musk + Chinese employee** -> frame chase as team need (not individual blame)
- **Grove + Indonesian employee** -> OKR discussion embedded in relationship context

## Communication Patterns by Directness

### High Directness (default, singapore, usa)
- State issues directly
- Use data and metrics
- Feedback can be given in group settings
- Action items are explicit

### Medium Directness (malaysia, china, india)
- Soften negative feedback with positive context
- Use "we" framing instead of "you"
- Prefer 1:1 for sensitive topics
- Allow face-saving options

### Low Directness (philippines, indonesia, srilanka)
- Always start with relationship warmth
- Never single out individuals publicly
- Use indirect language for criticism
- Provide private channels for difficult conversations

## Hierarchy Adaptation

### High Hierarchy (philippines, indonesia, srilanka, china, india)
- Respect reporting chains
- Senior team members addressed with appropriate titles
- Decisions communicated top-down
- Feedback flows through appropriate channels

### Low Hierarchy (default, usa)
- Open-door communication
- Direct access to leadership
- Merit-based recognition regardless of tenure
- Flat communication structure
